■UBS 曲線を求める

４つの制御点 (n = 4) 、で３次曲線 (M = 4) を表現する場合の、UBS 曲線を求めます。 ノットベクトル、 

T = [-3 -2 -1 0 1 2 3],

に、基底関数の定義式 

            t  -  ti              ti+M -  t
Ni,M(t) = ――――― Ni,M-1(t) + ―――――― Ni+1,M-1 (t),
           ti+M-1 - ti             ti+M - ti+1

を代入します。 まず、M = 1 の場合、ti ≦ t < ti+1の t しか値は残らないので、 

            t  -  t0              t0+1 -  t
N0,1(t) = ――――― N0,1-1(t) + ―――――― N0+1,1-1 (t),
           t0+1-1 - t0             t0+1 - t0+1
            t  -  t0              t0+1 -  t
        = ――――― N0,1-1(t) + ―――――― N0+1,1-1 (t),
           t0+1-1 - t0             t0+1 - t0+1

になります。
